# Demos

Generated chart examples using the EventEmbeddingDetector (exp 45). These are slightly cherry-picked for good examples — results vary by song and density settings.

All charts were generated with:
```bash
python detection_inference.py --checkpoint runs/detect_experiment_45/checkpoints/best.pt --density-mean 6.75 --density-peak 11.1 --andlaunch --hop-ms 100 --slide-frames 2 --gif gawr-gura-hololive.gif --gif-cycles 2
```
